# Менеджер контактов

Простое приложение для управления контактами с категориями, поиском и импортом/экспортом.

## Функциональность
- Добавление, редактирование, удаление контактов
- Категоризация: Друзья, Семья, Работа
- Поиск по имени/телефону/email
- Фильтр по категории
- Импорт/экспорт в CSV

## Технологии
- Python 3.10+
- PyQt 
- SQLite3 (база данных)
- csv (импорт/экспорт)
