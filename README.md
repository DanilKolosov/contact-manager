# Менеджер контактов

Простое приложение для управления контактами с категориями, поиском и импортом/экспортом.

## Функциональность
- Добавление, редактирование, удаление контактов
- Категоризация: Друзья, Семья, Работа
- Поиск по имени/телефону/email
- Фильтр по категории
- Импорт/экспорт в CSV

## Технологии
- Python 3.10+
- Tkinter (GUI)
- SQLite3 (база данных)
- csv (импорт/экспорт)

## Архитектура
Layered Architecture + Repository Pattern + Service Layer (см. раздел "Проектирование" в отчёте).

python main.py
